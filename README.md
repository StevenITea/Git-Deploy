# 🌐 Git Deploy - Tutorial

This project is a beginner-friendly tutorial designed to teach how to create and deploy a simple website using GitHub Pages.

## 🎯 Objective
Learn how to:
- Create a basic website using HTML and CSS
- Use Git commands (init, add, commit, push)
- Link a local project to a GitHub repository
- Deploy a website online with GitHub Pages

## 🧰 Technologies Used
- HTML5
- CSS3
- Git
- GitHub Pages

## 📚 What you will learn
This tutorial focuses on the fundamentals of deployment without using advanced frameworks. It is ideal for beginners who want to understand how a website goes live.
